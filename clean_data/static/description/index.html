Mass Clear Data(Clean data)

Main Features :
This module helps to user clear unwanted testing data using wizard on single click of button.

Path : Settings > Clean Data > Clean Data.

Easily clear following object data using boolean check true.
1.Sales
2.Delivery Orders
3.Purchase
4.All Transfers
5.Invoicing
6.Payments
7.All Journal Entries
8.Customers & Vendors
9.Chart Of Accounts,
User can also clean all above object using single click checked True all data boolean.

Notes: Do not use this module in production environment,
because using this module data clean/clear easily but restore is very difficult,
So do not use this module in production environment.
This module support only core object of the odoo not for custom objects.
