Overview

ReportMate extracts all the needed information for all the fields of any Odoo Model and displays complex data in a single view. Customers can create powerful, richly formatted, dynamic reports in the Tree, List, Graph, and Pivot format. This app allows to display the data in a flat data structure meaning, no more multiple clicks to arrive at your needed results. Increase your business productivity and save your developer's time by using ReportMate.

PROBLEM:

In Odoo, there is no option available for users to create custom reports on their own from the view.

SOLUTION:

The app ReportMate allows users to display all their records in a flat data structure. The company can now easily group several data sets to be displayed

Key Features

  • Create Custom View/Report based on Selected Model and by Selecting Fields/Relations on their own.
  • Reports can be displayed in Pivot and Graph View.
  • Created Reports can only be viewed by Groups that have access to view created reports/views.
  • Users can customize which fields to show in Search Filter/Group Filter.

    • Create/Configure Reports :
    • You can create Reports from the ReportMate menu as shown below:
    • While creating the Report, the user needs to select the model on which they wish to create their report.
    • After selecting the model, the user will be able to see the list view at the bottom where they can add the Report Field line one by one. While creating the Report Field line, users can:
    • Set the name of the field (display name)
    • Select the field / field chain of the selected Model. Users can also set up the field chain as deep and long as they need to
    • Set the field to be shown in Search and Group Filters
    • After adding all Report Field Lines, users can fill in other Report related information. While creating the Report user can :
    • Set the Menu Name (Report Menu Name) and select under which menu it will be shown
    • Set Menu Priority to decide the position ordering of Menu
    • Set Menu as active or inactive by ticking the checkbox to enable/disable menu at any time
    • Set Groups on Report through Group Access to control who will be allowed to view the created Report
    • Set Pivot and Graph options to enable Pivot and Graph views for Reports
    • After saving the report Users will have to refresh their browser to see the Menu for the report.

    • ReportMate Manager role:
    • Since ReportMate create reports using database queries, it can bypass all model securities to show the report data. Therefore there is security group for ReportMate.
    • ReportMate Manager role control who can Read/Create/Edit/Delete the ReportMate Records (Reports).
    • Admin is by default assigned as ReportMate Manager. They can give this role to other users by going into the User form page and setting up ReportMate Manager as True.
